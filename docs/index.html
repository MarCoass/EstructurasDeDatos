Conceptos basicos

Tipo de Dato: conjunto de valores que la variable puede tomar.
Tipo de Dato Abstracto (TDA): define un modelo de un tipo de datos, indicando su posible contenido y operaciones, pero no dice como debe implementarse.
Estructuras de datos: es una coleccion de variables conectadas de varias formas. El contenido basico es la "celda" o "nodo" que es capaz de almacenar un dato de un tipo determinado.

Clasificacion de las estructuras de datos

Estructuras lineales: guardan los datos uno detras de otro. Tienen operaciones para a√±adir, quitar y buscar elementos.
Estructuras jerarquicas: son estructuras jerarquizadas llamadas arbol, constituidas por nodos. Cada nodo tiene una relacion de parentesco.
Grafos: pueden expresar relaciones de conexion sin restriccion. Los elementos son llamados vertices o nodos y estan unidos por aristas o arcos.
